# ios_module_TapToCall
Tap to Call module allows users to make calls on predefined phone number.

**XML Structure declaration**

Tags:
- title - button name
- phone - phone number

Example:


    <data>
      <title><![CDATA[Button]]></title>  
      <phone><![CDATA[+12(345)1234123412]]></phone>
    </data>
